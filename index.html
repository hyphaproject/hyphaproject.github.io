Hyphaproject

Hyphaproject is a framework for home automation written in c++;

Subprojects

The project consists of some subprojects:

hypha

Hypha is the core library.

hypharunner

Hypharunner is a service to execute hypha handlers and hypha plugins.

hyphaplugins

Hyphaplugins are modules to read from sensors or send data to actors.

hyphahandlers

Hyphahandlers are modules with the logic that lets the plugins interact in a specific way.

hyphamanager

Hyphamanager is a desktop application used to configure connected modules.
